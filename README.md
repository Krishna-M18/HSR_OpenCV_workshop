# 🖥️ Computer Vision with Python Workshop

> A complete collection of projects developed during the **5-Day Offline Computer Vision Workshop** conducted by **Innomatics Research Labs**.

---

## 📖 Overview

This repository showcases my hands-on learning journey in Computer Vision using **Python**, **OpenCV**, and **YOLO (Ultralytics)**.

Throughout this workshop, I implemented real-world computer vision applications including:

- Image Processing
- Image Transformations
- Video Processing
- Face Detection
- Face Mask Detection
- YOLO Object Detection
- YOLO Instance Segmentation
- YOLO Pose Estimation
- Real-time Webcam Detection
